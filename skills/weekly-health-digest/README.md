# 每周健康周报 (Weekly Health Digest)

## 概述
聚合 health-memory 中最近 7 天的每日数据和所有纵向指标文件，生成叙事性健康周报，计算综合健康评分（0-100分），发现跨域关联，并输出下周可操作建议。

## 适用场景
- 每周结束时自动生成周报
- 用户主动要求查看健康总结
- 用户询问"这周健康状况怎么样"
- 需要回顾过去一周的综合健康数据

## 使用方式
- 自动触发: 每周日或周一自动生成上周周报
- 由场景技能调用: 可被其他场景技能调用以提供周度汇总数据
- 用户手动触发: 用户可指定特定周，如 "生成 2026-W10 周报"

## 输入格式
- 无输入或空: 默认生成最近完整一周的周报
- 指定周: "2026-W10" 或 "上周"
- 仅评分: "评分" 或 "只看分数" -- 仅输出健康评分和明细

## 输出示例
输出完整叙事性周报，包含：
- 健康评分 (0-100) 和评分明细表
- 各领域叙事性总结（血压、血糖、运动、睡眠等）
- 跨域关联发现（如运动与血压的关系）
- 本周亮点和里程碑
- 3-5 条具体的下周行动建议
- 数据完整性表格

## 数据存储
- 周报文件: `memory/health/weekly/YYYY-Wnn.md`
- 每日文件摘要: 写入周日的 `memory/health/daily/YYYY-MM-DD.md`

## 医学免责声明
本技能仅供健康参考，不构成医疗建议。如有健康问题，请咨询专业医疗人员。
