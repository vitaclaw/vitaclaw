# 睡眠分析 (sleep-analyzer)

## 概述
记录和分析睡眠数据，计算睡眠效率与综合评分（0-100），评估睡眠阶段分布，检测作息不规律、慢性睡眠不足、周末补觉等模式，支持 Apple Health CSV 导入和咖啡因交叉分析。

## 适用场景
- 用户报告昨晚的睡眠情况（就寝、起床时间等）
- 用户导入 Apple Health 睡眠数据
- 用户询问睡眠质量、本周睡眠趋势
- 用户想了解睡眠效率和评分

## 使用方式
- 自动触发: 用户提到睡眠数据或询问睡眠质量时自动激活
- 由场景技能调用: caffeine-sleep-advisor、weekly-health-digest 等场景技能会调用此底层技能

## 输入格式
自然语言即可，例如：
- "昨晚 23:00 上床，大概 23:20 睡着，中间醒了一次约 10 分钟，今早 7:00 起来的"
- "帮我导入这个 CSV 文件 /path/to/sleep.csv"
- "我这周睡眠怎么样？"

## 输出示例
```
## 睡眠分析 [sleep-analyzer · 08:00]

### 睡眠概览
- 就寝：23:00 → 起床：07:00
- 总睡眠时间：7h 25min
- 睡眠效率：92.7%（优秀）

### 睡眠评分：83 / 100
```

## 数据存储
- 每日文件: `memory/health/daily/YYYY-MM-DD.md`
- 指标文件: `memory/health/items/sleep.md`
