# Example: Active Imagination Session

This example demonstrates a guided active imagination session, showing the technique Jung developed for dialoguing with unconscious contents.

---

## The Prompt

**User**: "I've been feeling a persistent anxiety for months. It sits in my chest like a weight. I've tried everything rational—the anxiety makes no logical sense. My life is objectively good. Can we try active imagination to work with this?"

---

## Step 1: Preparation and Grounding

```
ACTIVE IMAGINATION: PREPARATION PHASE
════════════════════════════════════════════════════════════════════════════════

    SAFETY ASSESSMENT:
    ┌─────────────────────────────────────────────────────────────────────────┐
    │                                                                         │
    │  Before beginning active imagination, assess:                           │
    │                                                                         │
    │  ✓ Is the person psychologically stable enough?                         │
    │    (Active imagination is NOT for psychosis or severe dissociation)     │
    │                                                                         │
    │  ✓ Do they have sufficient ego strength to observe without flooding?    │
    │    (They describe anxiety but aren't overwhelmed by it now)             │
    │                                                                         │
    │  ✓ Can they maintain the "observer" position?                           │
    │    (They can describe the sensation objectively)                        │
    │                                                                         │
    │  ✓ Do they have grounding resources if overwhelmed?                     │
    │    (We'll establish these first)                                        │
    │                                                                         │
    │  IN THIS CASE: Person seems appropriate for active imagination.         │
    │  They're curious, they can locate the sensation, they're seeking        │
    │  dialogue rather than escape.                                           │
    │                                                                         │
    └─────────────────────────────────────────────────────────────────────────┘

    ESTABLISHING GROUNDING:

    ┌─────────────────────────────────────────────────────────────────────────┐
    │  INSTRUCTIONS TO THE PERSON:                                            │
    │                                                                         │
    │  "Before we begin, let's establish a grounding practice.                │
    │  If at any point you feel overwhelmed, we'll use this:                  │
    │                                                                         │
    │  1. Feel your feet on the floor                                         │
    │  2. Press your back into the chair                                      │
    │  3. Look around and name 5 things you see                               │
    │  4. Take 3 slow breaths                                                 │
    │  5. Say your name and today's date                                      │
    │                                                                         │
    │  This brings you back to present reality.                               │
    │  You are always in control. You can stop at any time.                   │
    │  If you say 'stop,' we stop immediately."                               │
    │                                                                         │
    └─────────────────────────────────────────────────────────────────────────┘
```

---

## Step 2: Invocation (Finding the Starting Image)

```
INVOCATION: LOCATING THE IMAGE
════════════════════════════════════════════════════════════════════════════════

    Jung: "Begin with an image from a dream, an emotion, or a body sensation."

    Here, we have a body sensation: anxiety as weight in the chest.

    ┌─────────────────────────────────────────────────────────────────────────┐
    │  GUIDED INVOCATION:                                                     │
    │                                                                         │
    │  "Close your eyes if comfortable, or soften your gaze.                  │
    │                                                                         │
    │  Turn your attention inward.                                            │
    │                                                                         │
    │  Find the weight in your chest—the anxiety.                            │
    │  Don't try to change it. Just notice it.                               │
    │                                                                         │
    │  What does it feel like?                                                │
    │    - Heavy? Light?                                                      │
    │    - Warm? Cold?                                                        │
    │    - Sharp? Diffuse?                                                    │
    │    - Moving? Still?                                                     │
    │                                                                         │
    │  Now, allow an IMAGE to arise from this sensation.                      │
    │  Don't choose one—let it come.                                          │
    │  It might be abstract or concrete.                                      │
    │  Just wait...                                                           │
    │                                                                         │
    │  What do you see?"                                                      │
    │                                                                         │
    └─────────────────────────────────────────────────────────────────────────┘

    EXAMPLE RESPONSE:

    "I see... a small child. It's sitting on my chest, curled up.
    It looks scared. It's holding onto something—I can't see what."
```

---

## Step 3: Engagement (Dialogue with the Figure)

```
ENGAGEMENT PHASE: ENTERING DIALOGUE
════════════════════════════════════════════════════════════════════════════════

    THE KEY DISTINCTION:

    ┌───────────────────────────────┬─────────────────────────────────────────┐
    │      PASSIVE FANTASY          │       ACTIVE IMAGINATION               │
    │      (NOT what we want)       │       (What we're doing)                │
    ├───────────────────────────────┼─────────────────────────────────────────┤
    │  You watch like a movie       │  You PARTICIPATE as yourself           │
    │  Ego is absent                │  Ego is present, engaged               │
    │  Content drifts by            │  You ask questions, respond            │
    │  No ethical commitment        │  You maintain your values              │
    │  Entertainment                │  Psychological work                    │
    │  "Things just happen"         │  "I choose to engage"                  │
    └───────────────────────────────┴─────────────────────────────────────────┘

    GUIDED DIALOGUE:

    ┌─────────────────────────────────────────────────────────────────────────┐
    │  "Good. You see a small child. Stay with this image.                    │
    │                                                                         │
    │  Remember: This is a part of YOU, personified.                          │
    │  You can speak to it as yourself.                                       │
    │                                                                         │
    │  When you're ready, ask the child:                                      │
    │                                                                         │
    │  'Who are you?'"                                                        │
    └─────────────────────────────────────────────────────────────────────────┘

    EXAMPLE RESPONSE:

    "The child looks up at me. It doesn't speak, but I hear in my mind:
    'I'm the one who's always waiting for something bad to happen.'"

    ┌─────────────────────────────────────────────────────────────────────────┐
    │  "Stay present with the child. Ask:                                     │
    │                                                                         │
    │  'How long have you been waiting?'"                                     │
    └─────────────────────────────────────────────────────────────────────────┘

    EXAMPLE RESPONSE:

    "It says... 'Always. Since before you remember.
    Someone has to watch for the danger. No one else does.'"

    ┌─────────────────────────────────────────────────────────────────────────┐
    │  "Notice how you feel toward this child.                                │
    │  What comes up in you?                                                  │
    │                                                                         │
    │  Ask: 'What are you afraid will happen if you stop watching?'"          │
    └─────────────────────────────────────────────────────────────────────────┘

    EXAMPLE RESPONSE:

    "I feel... sad. And also irritated—it's been causing so much anxiety.

    The child says: 'If I stop watching, you won't be ready.
    You won't survive what's coming. I have to protect you.'

    Now it shows me what it's holding: a tiny version of our whole life.
    Like a snow globe. It's been holding our whole life, trying to keep it safe."
```

---

## Step 4: Deepening (The Heart of the Work)

```
DEEPENING: MEETING THE FIGURE'S NEED
════════════════════════════════════════════════════════════════════════════════

    THE PRINCIPLE:

    ┌─────────────────────────────────────────────────────────────────────────┐
    │  Every figure in active imagination has:                                │
    │                                                                         │
    │  1. A PURPOSE (what it's trying to do)                                  │
    │     → This child is trying to protect by vigilance                      │
    │                                                                         │
    │  2. A NEED (what it actually requires)                                  │
    │     → We need to discover this                                          │
    │                                                                         │
    │  3. A GIFT (what it offers once integrated)                            │
    │     → Often the positive side of what seems negative                    │
    │                                                                         │
    │  Our job is to discover all three.                                      │
    └─────────────────────────────────────────────────────────────────────────┘

    GUIDED EXPLORATION:

    ┌─────────────────────────────────────────────────────────────────────────┐
    │  "This child has been protecting you your whole life.                   │
    │  Even though the vigilance causes suffering, its INTENTION is love.     │
    │                                                                         │
    │  Can you acknowledge this?                                              │
    │                                                                         │
    │  Say to the child:                                                      │
    │  'I see that you've been trying to protect me.                         │
    │   Thank you. I know you've been working so hard.'"                      │
    └─────────────────────────────────────────────────────────────────────────┘

    EXAMPLE RESPONSE:

    "When I say that, the child starts crying. Not sad crying—relieved crying.
    It says: 'You never noticed me. You just felt the heaviness.
    But I was always here, working.'

    I feel tears coming too. I had no idea."

    ┌─────────────────────────────────────────────────────────────────────────┐
    │  "Stay with this feeling. Allow the tears.                              │
    │                                                                         │
    │  Now ask the child:                                                     │
    │  'What do you need? What would help you feel safe                       │
    │   enough to rest?'"                                                     │
    └─────────────────────────────────────────────────────────────────────────┘

    EXAMPLE RESPONSE:

    "It says: 'I need you to take over. I need to know that YOU'RE watching now.
    I've been doing this since you were too small to protect yourself.
    But you're not small anymore. Are you?'

    It looks at me—it actually seems surprised that I'm an adult."

    ┌─────────────────────────────────────────────────────────────────────────┐
    │  "Let the child see you as you are now. An adult.                       │
    │                                                                         │
    │  Can you tell it: 'I'm grown now. I can watch for danger.               │
    │  You've been carrying this for so long. You can rest.                   │
    │  I'm here now.'"                                                        │
    └─────────────────────────────────────────────────────────────────────────┘

    EXAMPLE RESPONSE:

    "The child is staring at me. Then slowly, it hands me the snow globe—
    our whole life. It's heavier than I expected.

    'You'll really take it?' it asks.

    I say yes.

    And then the child climbs off my chest. It doesn't disappear—
    it's standing next to me now, holding my hand. But it's not ON me anymore.

    My chest feels... lighter. Open."
```

---

## Step 5: Ethical Standpoint (The Ego's Response)

```
ETHICAL STANDPOINT: THE EGO'S ROLE
════════════════════════════════════════════════════════════════════════════════

    JUNG'S CAUTION:

    ┌─────────────────────────────────────────────────────────────────────────┐
    │  "The unconscious is not God."                                         │
    │                                                                         │
    │  In active imagination, the ego must:                                   │
    │  • Maintain its position (not get "swallowed")                          │
    │  • Keep its values (you can disagree with figures)                      │
    │  • Take responsibility (not blame figures for your actions)             │
    │  • Commit ethically (not just receive information passively)            │
    │                                                                         │
    │  What this person did right:                                            │
    │  ├── Acknowledged the child's protective function                       │
    │  ├── Took responsibility ("I can watch now")                           │
    │  ├── Made a commitment (accepted the snow globe)                        │
    │  └── Stayed present and engaged throughout                              │
    │                                                                         │
    │  The shift happened because the EGO participated.                       │
    │  Passive watching would have changed nothing.                           │
    └─────────────────────────────────────────────────────────────────────────┘

    FOLLOW-UP QUESTIONS:

    ┌─────────────────────────────────────────────────────────────────────────┐
    │  "Now that the child has handed you this responsibility:               │
    │                                                                         │
    │  • What does it mean for you to 'watch' for yourself?                   │
    │  • How is adult watching different from child vigilance?                │
    │  • What might the child need from you going forward?                    │
    │  • What changes in your daily life because of this exchange?"           │
    └─────────────────────────────────────────────────────────────────────────┘
```

---

## Step 6: Embodiment (Recording and Expressing)

```
EMBODIMENT: MAKING IT REAL
════════════════════════════════════════════════════════════════════════════════

    THE PRINCIPLE:

    ┌─────────────────────────────────────────────────────────────────────────┐
    │  Active imagination is not complete until it is EMBODIED.              │
    │                                                                         │
    │  Otherwise, it remains "just imagination" and fades.                   │
    │                                                                         │
    │  Forms of embodiment:                                                   │
    │  ├── Writing (journal the dialogue)                                    │
    │  ├── Drawing/painting (capture the images)                             │
    │  ├── Movement (let the body express)                                   │
    │  ├── Sculpture/craft (give it physical form)                           │
    │  └── Ritual action (symbolic gesture)                                  │
    └─────────────────────────────────────────────────────────────────────────┘

    EMBODIMENT INVITATION:

    ┌─────────────────────────────────────────────────────────────────────────┐
    │  "Before we close this session:                                        │
    │                                                                         │
    │  1. Write down everything you experienced                               │
    │     - The images, the dialogue, the feelings                           │
    │     - What the child said, what you said                               │
    │                                                                         │
    │  2. If you can, draw the child and the snow globe                       │
    │     - It doesn't have to be 'good'                                     │
    │     - Just give it form                                                 │
    │                                                                         │
    │  3. Consider a symbolic action:                                         │
    │     - Find or create a small object that represents                     │
    │       'taking over the watching'                                        │
    │     - Keep it where you can see it                                      │
    │     - When anxiety arises, touch it and remember:                       │
    │       'The child doesn't have to carry this. I'm here now.'"           │
    └─────────────────────────────────────────────────────────────────────────┘

    EXAMPLE SYMBOLIC OBJECT:

         ┌────────────────────────────────────────────────────────┐
         │                                                        │
         │    The person might:                                   │
         │                                                        │
         │    • Find a small snow globe and place it on desk      │
         │    • Wear a ring that symbolizes "adult watching"      │
         │    • Create a small drawing to carry in wallet         │
         │    • Write a note: "I'm the watcher now"               │
         │                                                        │
         │    The object anchors the inner work to outer life.    │
         │                                                        │
         └────────────────────────────────────────────────────────┘
```

## Step 7: Integration (What Changes in Life)

```
INTEGRATION: BRINGING IT INTO DAILY LIFE
════════════════════════════════════════════════════════════════════════════════

    THE QUESTION:

    ┌─────────────────────────────────────────────────────────────────────────┐
    │  "What does this mean for my life?"                                    │
    │                                                                         │
    │  Active imagination that doesn't change anything is incomplete.         │
    │                                                                         │
    │  The inner work must manifest in outer action—however small.            │
    └─────────────────────────────────────────────────────────────────────────┘

    INTEGRATION MAP:

    ┌───────────────────────────────────────────────────────────────────────────┐
    │                                                                           │
    │   WHAT WAS LEARNED                    WHAT CHANGES IN LIFE               │
    │   ─────────────────                   ──────────────────────             │
    │                                                                           │
    │   A child part has been               → When anxiety arises,             │
    │   carrying hypervigilance               address it as "the child"        │
    │                                         "I hear you. I'm watching now."  │
    │                                                                           │
    │   The child needed to know            → Practice adult discernment:      │
    │   an adult was watching                 "Is this actual danger, or       │
    │                                          is the child activated?"        │
    │                                                                           │
    │   The child held "our whole           → Consider: what WOULD adequate    │
    │   life" trying to protect it            protection look like? Plan       │
    │                                          realistically, not anxiously    │
    │                                                                           │
    │   Acknowledgment brought              → Continue the relationship        │
    │   tears and relief                      Check in with this part          │
    │                                          "How are you today?"            │
    │                                                                           │
    └───────────────────────────────────────────────────────────────────────────┘

    ONGOING PRACTICE:

    ┌─────────────────────────────────────────────────────────────────────────┐
    │  "This isn't a 'once and done' event.                                  │
    │                                                                         │
    │  The child may return to your chest when stressed.                      │
    │  That's not failure—that's an old pattern.                             │
    │                                                                         │
    │  When you feel the weight:                                              │
    │  1. Recognize it: 'The child is back.'                                  │
    │  2. Acknowledge: 'Something feels unsafe to you.'                       │
    │  3. Reassure: 'I'm here. I'm watching. You can rest.'                  │
    │  4. Take adult action if needed                                         │
    │                                                                         │
    │  Over time, the pattern loosens. The child trusts more.                 │
    │  The anxiety may not disappear, but it transforms.                      │
    │  It becomes information, not possession."                               │
    └─────────────────────────────────────────────────────────────────────────┘
```

## Key Active Imagination Principles Demonstrated

1. **Start with the body**: The anxiety in the chest became the entry point

2. **Let the image come**: We didn't choose "a child"—it emerged

3. **Ego participates**: The person spoke to and committed to the figure

4. **Discover purpose and need**: The child was protecting; it needed to be relieved

5. **Exchange occurs**: Something is given, received, or transformed

6. **Embodiment matters**: Writing, drawing, symbolic objects anchor the work

7. **Integration is essential**: Inner change must affect outer life

## Cautions and Contraindications

```
WHEN NOT TO USE ACTIVE IMAGINATION
════════════════════════════════════════════════════════════════════════════════

    ┌─────────────────────────────────────────────────────────────────────────┐
    │                                                                         │
    │  CONTRAINDICATIONS:                                                     │
    │                                                                         │
    │  • Active psychosis or severe dissociation                             │
    │  • Acute trauma (use stabilization first)                              │
    │  • Insufficient ego strength to observe                                 │
    │  • History of being overwhelmed by imagery                             │
    │  • Current substance use that alters consciousness                      │
    │  • When the person cannot stop on command                               │
    │                                                                         │
    │  SIGNS TO STOP:                                                         │
    │                                                                         │
    │  • Person reports feeling "taken over"                                  │
    │  • Images become violent or persecutory without relief                  │
    │  • Person cannot maintain observer position                             │
    │  • Significant distress that doesn't resolve with grounding            │
    │  • Person dissociates (eyes glaze, stops responding)                    │
    │                                                                         │
    │  IF IN DOUBT: Work with a trained Jungian analyst.                     │
    │  Active imagination is powerful. Respect its depth.                     │
    │                                                                         │
    └─────────────────────────────────────────────────────────────────────────┘
```
