# 睡眠优化建议 (sleep-optimizer)

## 概述
基于睡眠分析数据、咖啡因摄入数据和生活习惯因素（屏幕时间、运动、卧室环境、压力等），生成按优先级排序的个性化睡眠改善建议和 7 天改善计划。

## 适用场景
- sleep-analyzer 识别出睡眠问题后，用户希望获得改善建议
- 用户主动询问如何改善睡眠
- 用户询问某种补充剂（如褪黑素）是否有效
- 用户提供生活习惯信息希望获得针对性建议

## 使用方式
- 自动触发: 用户询问如何改善睡眠或请求睡眠优化建议时激活
- 由场景技能调用: caffeine-sleep-advisor 等综合场景技能会调用此技能生成优化方案

## 输入格式
自然语言即可，例如：
- "帮我分析一下怎么改善睡眠"
- "我最近睡不好，睡前经常刷手机到 12 点，压力也很大"
- "吃褪黑素有用吗？"

## 输出示例
```
# 睡眠优化建议报告

## 识别的问题（按优先级）
### P1 - 咖啡因摄入时间偏晚
### P2 - 入睡潜伏期偏长
### P3 - 作息不够规律

## 个性化建议
1. [高优先级] 咖啡因截止时间提前到 14:00
2. [中优先级] 建立 30 分钟睡前放松程序
3. [中优先级] 固定 07:00 起床时间

## 7 天改善计划
...
```

## 数据存储
- 本技能为只读分析技能，不写入 health-memory
- 读取来源: `memory/health/items/sleep.md`、`memory/health/items/caffeine.md`、`memory/health/daily/*.md`

## 医学免责声明
本技能仅供健康参考，不构成医疗建议。如有健康问题，请咨询专业医疗人员。
