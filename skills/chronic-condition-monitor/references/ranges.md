# Reference Ranges (Chinese Clinical Guidelines)

## Normal Range Reference Table

| Indicator | Normal Range | Caution Range | Critical Value |
|-----------|-------------|---------------|----------------|
| SBP (Systolic BP) | < 120 mmHg | 120-139 (high-normal) | >= 180 (hypertensive crisis) |
| DBP (Diastolic BP) | < 80 mmHg | 80-89 (high-normal) | >= 110 (hypertensive crisis) |
| FPG (Fasting Glucose) | 3.9-6.1 mmol/L | 6.2-6.9 (impaired fasting glucose) | >= 16.7 or <= 2.8 |
| 2hPG (Postprandial Glucose) | < 7.8 mmol/L | 7.8-11.0 (impaired glucose tolerance) | >= 16.7 |
| HbA1c | < 6.0% | 6.0-6.4% (prediabetes) | >= 9.0% |
| BMI | 18.5-23.9 | 24.0-27.9 (overweight) | >= 28 (obese) |
| Waist (male) | < 90 cm | 90-95 cm | > 95 cm |
| Waist (female) | < 85 cm | 85-90 cm | > 90 cm |
| Uric Acid (male) | < 420 umol/L | 420-480 | > 540 |
| Uric Acid (female) | < 360 umol/L | 360-420 | > 480 |
| Creatinine (male) | 57-111 umol/L | 111-133 | > 177 |
| Creatinine (female) | 41-81 umol/L | 81-106 | > 177 |
| eGFR | >= 90 mL/min/1.73m^2 | 60-89 (mild decrease) | < 30 (severe decrease) |
| TC (Total Cholesterol) | < 5.2 mmol/L | 5.2-6.2 (borderline high) | > 7.2 |
| TG (Triglycerides) | < 1.7 mmol/L | 1.7-2.3 (borderline high) | > 5.6 (very high) |
| LDL-C | < 3.4 mmol/L | 3.4-4.1 (borderline high) | > 4.9 |
| HDL-C | > 1.0 mmol/L | 0.8-1.0 (low) | < 0.8 |
| ALT | < 40 U/L | 40-80 (mild elevation) | > 200 |
| AST | < 40 U/L | 40-80 (mild elevation) | > 200 |

## Classification Logic

- Within normal range -> Normal
- Within caution range -> Elevated/Low, needs attention
- Reaches critical value -> Immediate alert, advise medical attention

## Metabolic Syndrome Criteria (Chinese Guidelines)

Diagnose when 3 of 5 criteria are met:

| # | Criterion | Threshold |
|---|-----------|-----------|
| 1 | Central obesity | Waist: male >= 90 cm, female >= 85 cm |
| 2 | Elevated triglycerides | TG >= 1.7 mmol/L |
| 3 | Low HDL-C | Male < 1.0 mmol/L, female < 1.3 mmol/L |
| 4 | Elevated blood pressure | SBP >= 130 or DBP >= 85 mmHg |
| 5 | Elevated fasting glucose | FPG >= 6.1 mmol/L |

## Diabetes + Hypertension Comorbidity Thresholds

- Diabetes: FPG >= 7.0 mmol/L or HbA1c >= 6.5%
- Hypertension: SBP >= 140 or DBP >= 90 mmHg
- Combined target BP: < 130/80 mmHg

## CKD Staging (eGFR-based)

| Stage | eGFR (mL/min/1.73m^2) | Description |
|-------|------------------------|-------------|
| G1 | >= 90 | Normal or high |
| G2 | 60-89 | Mildly decreased |
| G3a | 45-59 | Mildly to moderately decreased |
| G3b | 30-44 | Moderately to severely decreased |
| G4 | 15-29 | Severely decreased |
| G5 | < 15 | Kidney failure |

## Severity Grading Criteria

**Level 1 (Mild)**: 1 indicator mildly out of range (caution zone), no worsening trend

**Level 2 (Moderate)**: 2+ indicators out of range, OR 1 significantly elevated (beyond caution, below critical), OR progressive worsening detected

**Level 3 (Severe)**: Any indicator at critical value, OR 3+ simultaneously out of range, OR rapid change (> 20%), OR metabolic syndrome detected
