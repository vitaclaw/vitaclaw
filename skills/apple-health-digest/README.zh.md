# 🍎 Apple 健康数据分析（apple-health-digest）

> 解析 Apple Health 导出数据，与治疗历史对照，揭示化疗/靶向治疗期间身体指标的变化规律。

## 适用人群

- 使用 Apple Watch 或 iPhone 记录健康数据的患者
- 想了解"化疗期间体重掉了多少"、"靶向治疗后心率有没有异常"的患者
- 需要向医生汇报近期活动能力、睡眠质量的患者

## 核心价值：医疗关联分析

与普通 Apple Health 分析工具的区别在于**将可穿戴数据与治疗历史对照**。

不只是显示"你走了多少步"，而是以每个治疗阶段（化疗周期、靶向治疗期）为时间窗口，分析：

- 化疗期间**体重变化**（下降 > 5% 自动预警）
- 化疗/靶向治疗期间**活动量下降**（日均步数变化，作为疲劳度的代理指标）
- 治疗期间**心率变化**（均值 > 100 bpm 预警）
- 每个治疗周期内**睡眠时长**的变化

## 导入方式

在 iPhone 上操作：**健康 App → 右上角头像 → 导出所有健康数据 → 分享 export.zip**

解压后得到 `export.xml`，告诉 Agent 文件路径即可：

```
"帮我导入Apple Health数据，文件在桌面的export.xml"
"分析我的苹果健康数据"
```

## 触发方式

```
"分析我的苹果健康数据"
"导入Apple Health数据"
"化疗期间体重变化怎么样？"
"治疗期间心率有没有异常？"
"看我的步数趋势"
"最近3个月的活动量"
```

## 支持的健康指标

| 指标 | 生成文件 | 内容 |
|------|---------|------|
| 心率 | `心率趋势.md` | 每日均值 / 最低 / 最高 |
| 血压 | `血压记录.md` | 收缩压 + 舒张压配对记录 |
| 血氧 SpO2 | `血氧记录.md` | 每次测量值 |
| 体重 | `体重变化.md` | 每次称重 + 趋势箭头 |
| 步数 | `步数记录.md` | 每日总步数 + 周均值 |
| 睡眠 | `睡眠记录.md` | 每日睡眠时长 |

## 生成的报告文件

解析完成后，在患者目录的 `09_Apple_Health/` 下生成：

```
09_Apple_Health/
├── 心率趋势.md
├── 血压记录.md
├── 血氧记录.md
├── 体重变化.md
├── 步数记录.md
├── 睡眠记录.md
└── 治疗相关性分析.md    ← 核心报告：按治疗方案分段，逐项对比指标变化
```

`治疗相关性分析.md` 示例摘要：

```
## 一线化疗（FOLFOX × 6 周期）2024-01-15 → 2024-04-20

| 指标       | 治疗前   | 治疗中   | 变化       |
|-----------|---------|---------|-----------|
| 体重       | 62 kg   | 57 kg   | ↓ 8.1% ⚠ |
| 日均步数   | 6,200   | 2,800   | ↓ 55%     |
| 静息心率   | 68 bpm  | 74 bpm  | ↑ 8.8%    |
| 平均睡眠   | 7.2 h   | 6.1 h   | ↓ 15%     |
```

## 与其他 Skill 的关系

- **依赖 medical-record-organizer**：读取其建立的患者目录和治疗历史（`06_治疗决策历史/`）来分段分析；需先用 medical-record-organizer 建档
- **联动 health-timeline**：导入完成后，health-timeline 的治疗阶段视图可叠加苹果健康数据
