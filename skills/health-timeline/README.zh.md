# 📅 健康时间线查询（health-timeline）

> 查询和可视化病情历史，追踪肿瘤标志物趋势，回顾每个治疗阶段的疗效。

## 适用人群

- 想快速回顾近几个月病情进展的患者或家属
- 准备复诊，需要整理指标趋势给医生看的患者
- 想对比不同化疗/靶向治疗周期效果的患者

## 功能特性

- **全时间线展示**：按年月倒序排列所有就医记录，包含诊断、治疗、检查等事件
- **肿瘤标志物趋势**：ASCII 图直观显示 CEA / CA-199 / AFP / CA-125 的升降，自动标注变化幅度
- **治疗阶段分组**：一线、二线、三线治疗各自汇总，含用药时间、方案、不良反应
- **疗效评估记录**：CR（完全缓解）/ PR（部分缓解）/ SD（疾病稳定）/ PD（疾病进展）
- **异常值标注**：超出参考范围的指标显示 ⚠ 异常
- **日期区间筛选**："2024年"、"最近3个月"、"从1月到6月"等自然语言直接查询
- **文档类型筛选**："所有CT报告"、"血常规结果"、"基因检测"等
- **打印版报告**：生成就诊时可打印的 Markdown 汇总
- **Apple Health 联动**：若已导入苹果健康数据，可在治疗期间叠加显示体重、心率、步数变化

## 触发方式

```
"看我的病情时间线"
"最近3个月CEA有没有升高？"
"化疗后指标变化怎么样？"
"查看治疗历史"
"帮我生成一份就诊摘要"
"二线治疗的疗效评估"
"疗效评估"
```

## 5 种查询模式

| 模式 | 说明 | 示例问法 |
|------|------|---------|
| 全时间线 | 按时间显示所有事件 | "看我的完整病情记录" |
| 指标趋势 | 某指标的历史变化 + ASCII 图 | "CEA变化趋势" / "最近肿瘤标志物" |
| 治疗阶段 | 按治疗线分组，含疗效评估 | "化疗期间的疗效" / "查看二线方案" |
| 日期区间 | 指定时间段内的记录 | "2024年的检查记录" / "最近6个月" |
| 文档类型 | 只看某类文档 | "所有CT报告" / "基因检测结果" |

## 系统要求

无额外软件依赖。

需先用 **medical-record-organizer** 为患者建立目录（含 `timeline.md`），本 skill 在此基础上读取数据。

## 与其他 Skill 的关系

- **依赖 medical-record-organizer**：读取其建立的 `timeline.md`、`INDEX.md` 和 `肿标趋势.md`，无此基础数据则无法使用
- **联动 apple-health-digest**：若已导入苹果健康数据，治疗阶段视图中可叠加体重 / 心率 / 步数变化
