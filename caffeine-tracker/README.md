# 咖啡因追踪 (caffeine-tracker)

## 概述
记录每日咖啡因摄入，使用半衰期衰减模型（t1/2=5.7h）估算体内残留量，预测安全入睡时间，并提供每日摄入统计与健康警告。

## 适用场景
- 用户报告喝了咖啡、茶、能量饮料等含咖啡因饮品
- 用户询问当前体内咖啡因残留量
- 用户想知道什么时候可以安全入睡
- 用户询问今日咖啡因摄入是否超标

## 使用方式
- 自动触发: 用户提到咖啡因饮品摄入或询问咖啡因相关问题时自动激活
- 由场景技能调用: caffeine-sleep-advisor 等综合场景技能会调用此底层技能

## 输入格式
自然语言即可，例如：
- "刚喝了一杯拿铁"
- "08:30 喝了一杯美式"
- "下午3点喝了一罐红牛"

## 输出示例
```
## 咖啡因摄入记录 [caffeine-tracker · 14:30]

已记录：拿铁 80mg (14:30)

### 当前状态
- 今日总摄入：230mg / 400mg
- 当前体内残留：约 185mg
- 预计安全入睡时间：22:45
```

## 数据存储
- 每日文件: `memory/health/daily/YYYY-MM-DD.md`
- 指标文件: `memory/health/items/caffeine.md`

## 医学免责声明
本技能仅供健康参考，不构成医疗建议。如有健康问题，请咨询专业医疗人员。
