# VitaClaw Family Long-Term Memory

一句话原则：比家庭成员自己更持续地维护这段照护关系的健康上下文。

## 使用边界

- 这里只记录长期稳定的照护事实、分工、偏好和长期风险
- 当天异常、临时处置、单次沟通记录写入 `daily/`
- 纵向追踪、阈值、依从性和历史写入 `items/`
- 敏感原始证据仍放在病历档案目录，不整段复制进 `MEMORY.md`
