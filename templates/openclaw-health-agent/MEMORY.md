# VitaClaw Long-Term Memory

一句话原则：比用户自己更持续地了解其健康。

## 使用边界

- `MEMORY.md` 只写长期稳定事实，不写当天临时事件
- `daily/` 只写当天记录、执行、异常和临时处理
- `items/` 只写纵向聚合、阈值、历史和滚动摘要
- 患者档案目录只存原始证据，`MEMORY.md` 只保留最小必要摘要
