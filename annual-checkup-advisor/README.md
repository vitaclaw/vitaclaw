# 年度体检深度顾问 (Annual Checkup Advisor)

## 概述

综合解读年度体检报告的场景编排技能。整合体检报告解析、实验室指标深度解读、逐年对比、家族史遗传风险分层、多基因风险评分、基因营养建议、中医体质辨识和临床指南查询，生成结构化的7章节年度健康深度报告。

## 适用场景

- 拿到年度体检报告，需要全面深度解读
- 需要对比历年体检数据，追踪指标变化趋势
- 希望结合家族病史进行遗传风险分层
- 拥有基因检测数据，希望获得多基因风险评分和营养基因建议
- 想获得中医体质辨识和生活方式建议

## 使用方式

- 手动调用: `/annual-checkup-advisor [体检报告文本或PDF路径]`
- 自动触发: 当用户上传体检报告或询问体检结果解读时

## 输入格式

| 输入项 | 必填 | 说明 |
|--------|------|------|
| 体检报告 | 是 | 文本或PDF格式的年度体检报告 |
| 往年体检报告 | 否 | 1份或多份往年报告，用于逐年对比 |
| 家族病史 | 是 | 一级亲属疾病信息（父母/兄弟姐妹/子女） |
| 基因检测数据 | 否 | 基因芯片或全基因组测序结果 |

## 输出示例

生成7章节年度健康深度报告，包括：检查结果总览（红/黄/绿分级）、逐项指标解读、逐年趋势对比、遗传风险提示（家族史+PRS）、中医体质与生活方式建议、行动计划（复查/就诊/生活调整）、下次体检重点关注。

## 调用的子技能

| 技能 | 用途 |
|------|------|
| checkup-report-interpreter | 解析体检报告，提取40+指标并异常分级 |
| lab-results | 异常指标深度临床解读 + 逐年对比 |
| patiently-ai | 将专业解读翻译为通俗中文 |
| family-health-analyzer | 家族史遗传风险分层 |
| gwas-prs | 多基因风险评分（可选，需基因数据） |
| nutrigx_advisor | 基因-营养个性化建议（可选，需基因数据） |
| tcm-constitution-analyzer | 中医体质辨识和生活方式建议 |
| tooluniverse-clinical-guidelines | 临床指南推荐查询 |
| deep-research | 文献深度研究（可选，针对严重异常） |

## 数据存储

- 本技能不直接调用 `health-memory` 进行数据持久化
- 如需长期追踪，建议将关键指标录入 `memory/health/items/` 下的对应文件（如 `blood-lipids.md`）

## 医学免责声明

本技能仅供健康参考，不构成医疗建议。如有健康问题，请咨询专业医疗人员。
