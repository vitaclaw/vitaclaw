# 用药提醒与依从性追踪 (Medication Reminder)

## 概述
管理多药记录，生成每日服药计划，统计依从率，识别漏服模式（时段/星期/药物维度），追踪药物余量和补充提醒。

## 适用场景
- 用户添加、停用或修改药物
- 用户记录每日服药情况
- 需要查看依从率统计和漏服模式分析
- 需要生成每日服药计划
- 需要药物补充提醒

## 使用方式
- 自动触发: 当用户提及服药、用药相关内容时自动调用
- 由场景技能调用: hypertension-daily-copilot、diabetes-control-hub 等场景技能可调用本技能

## 输入格式
自然语言输入，例如：
- "新增药物：阿托伐他汀 20mg 每日一次 晚餐后 共60片"
- "今天早上药都吃了，晚上的阿托伐他汀忘了"
- "查看本月依从率"
- "生成今天的服药计划"
- "停用阿托伐他汀"

## 输出示例
根据操作类型输出每日服药计划表、依从率统计报告（含漏服模式分析）、药物变更确认等。

## 数据存储
- 每日文件: `memory/health/daily/YYYY-MM-DD.md`
- 指标文件: `memory/health/items/medications.md`
- 健康档案: `memory/health/_health-profile.md`

## 医学免责声明
本技能仅供健康参考，不构成医疗建议。如有健康问题，请咨询专业医疗人员。
