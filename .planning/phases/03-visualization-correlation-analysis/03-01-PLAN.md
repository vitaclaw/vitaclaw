    **Module-level setup (per D-02):**
    - `matplotlib.use("Agg")` before any pyplot import
    - `from __future__ import annotations` at top
    - Module-level `_configure_cjk_fonts()` function that:
      - Scans `matplotlib.font_manager.fontManager.ttflist` for CJK fonts
      - Priority order: PingFang HK, PingFang SC, Heiti TC, Hiragino Sans GB, Hiragino Sans, STHeiti, SimHei, Noto Sans CJK SC, Arial Unicode MS
      - Sets `plt.rcParams["font.sans-serif"]` and `plt.rcParams["axes.unicode_minus"] = False`
      - Prints warning to stderr if no CJK font found
    - Call `_configure_cjk_fonts()` at module level

    **Class `HealthChartEngine`:**
    - `__init__(self, data_dir: str | None = None, person_id: str | None = None)`
    - Uses `CrossSkillReader` for data access (person_id aware)
    - Default figure size: (12, 6), DPI: 150

    **Method `generate_blood_pressure_chart(self, days: int = 30) -> str` (per D-06):**
    - Dual-line: systolic (red #E74C3C, "o-") + diastolic (blue #3498DB, "s-")
    - Reference bands via axhspan (per D-03): normal <120/80 green, elevated 120-129 gold, stage 1 130-139 orange, stage 2 >=140 red
    - Y-axis: 40-200 mmHg (per Pitfall 3, never auto-scale)
    - Title: f"血压趋势 (近{days}天)"
    - Labels: "收缩压", "舒张压"
    - Saves to `data/blood-pressure-tracker/charts/{metric}_{pid}_{days}d_{date}.png`
    - pid = person_id or "self" (per Pitfall 6)
    - Returns file path string, or "" if no data

    **Method `generate_blood_glucose_chart(self, days: int = 30) -> str` (per D-06):**
    - Single line with markers
    - Reference bands (per D-03): normal 3.9-6.1 green, pre-diabetic 6.1-7.0 gold, diabetic >7.0 red
    - Y-axis: 2-15 mmol/L
    - Title: f"血糖趋势 (近{days}天)"

    **Method `generate_weight_chart(self, days: int = 30) -> str` (per D-06):**
    - Line chart with data points + moving average trend line (7-day window via numpy.convolve)
    - No fixed reference bands (per D-03: trend only)
    - Y-axis: auto with 10% padding (per Pitfall 3)
    - Title: f"体重趋势 (近{days}天)"

    **Method `generate_sleep_chart(self, days: int = 30) -> str` (per D-06):**
    - Bar chart (duration per day in hours)
    - Reference band: 7-9h recommended range (per D-03)
    - Y-axis: 0-14 hours
    - Title: f"睡眠时长 (近{days}天)"

    **Common patterns for ALL chart methods:**
    - Use OO API: `fig, ax = plt.subplots()` (per Pitfall 2, never stateful plt.plot)
    - Always `plt.close(fig)` after savefig
    - `fig.autofmt_xdate(rotation=45)` for date labels
    - `fig.tight_layout()` before save
    - `ax.grid(True, alpha=0.3)`
    - Show data point count in subtitle: f"共 {n} 个数据点"
    - For sparse data: show markers always, connect with lines only when gap <= 3 days

    **Data access pattern:**
    - Use `CrossSkillReader` with person_id for all reads
    - Parse timestamps from records: `r["timestamp"][:10]` for date grouping
    - Handle missing/empty data gracefully — return "" with no file created
